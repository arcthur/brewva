# Common Rationalizations

| Excuse                                        | Reality                                                                  |
| --------------------------------------------- | ------------------------------------------------------------------------ |
| "Interest is enough for now"                  | Interest is not demand. Look for pain, workaround, urgency, or pull.     |
| "Broad market means bigger opportunity"       | Early breadth hides weak specificity. Narrow beats wide early.           |
| "A builder project does not need positioning" | It still needs delight, a showable artifact, and a first audience.       |
| "We can validate while building"              | Sometimes yes, but office-hours must name the cheapest validation first. |
| "Strategy can handle this"                    | Strategy reviews a wedge; office-hours diagnoses whether one exists.     |
| "Discovery can handle this"                   | Discovery frames an existing request; office-hours tests a new premise.  |
