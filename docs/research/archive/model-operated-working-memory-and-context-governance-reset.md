## Historical Summary

This archive note records the long-form RFC that drove the Phase A and Phase B
context reset. The accepted decision is now
`docs/research/decisions/model-operated-working-memory-and-context-governance-reset.md`.

The reset replaced runtime-owned cognitive routing with the constitutional line:

`Model owns attention. Kernel owns consequence. Tape owns truth. Runtime owns physics.`

The implemented cutover deleted or demoted the old default path:

- context-source provider registry and hosted injection orchestration
- hidden recall prompt admission and regex intent classification
- `skill_load` gates before investigation
- deterministic projection as primary compaction
- typed deliberation, optimization, and narrative memory artifacts
- pressure-service ownership of cognitive decisions

The follow-up validation tracker is archived at
`docs/research/archive/model-operated-working-memory-evaluation.md`. The
accepted ergonomic contract now lives in
`docs/research/decisions/model-operated-context-skills-and-memory-ergonomics.md`.
